Laravel Framework 10.21.1 & PHP 8.1
-----

1. composer install
2. cp .env.example .env
3. Configure .env .
4. php artisan key:generate
5. php artisan passport:install
6. php artisan migrate
7. php artisan storage:link
8. php artisan serve
